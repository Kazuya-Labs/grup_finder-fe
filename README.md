# WhatsApp Group Finder Tool

## Project Description
The WhatsApp Group Finder Tool is designed to help users easily find and join various WhatsApp groups based on their interests. This project aims to streamline the process of discovering relevant groups and foster community engagement.

## Features
- **Search Functionality:** Users can search for groups based on keywords related to their interests.
- **Categories:** Groups are categorized for easy navigation.
- **User Authentication:** Login and registration features for group administrators.
- **Group Management:** Admins can create, edit, and delete groups.
- **Responsive Design:** Mobile-friendly web application that works seamlessly on any device.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Kazuya-Labs/grup_finder-fe.git
   ```
2. Navigate into the project directory:
   ```bash
   cd grup_finder-fe
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm run dev
   ```

## Usage
- Once the application is running, users can access it via `http://localhost:3000`.
- Use the search bar to find groups or browse through categories.
- For group administrators, use the admin panel to manage your groups.

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Deployment:** Vercel for frontend; Heroku for backend
